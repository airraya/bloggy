This is Hexo theme named Bloggy

## Getting started

Install dependencies:

``` bash
$ git clone https://github.com/airraya/bloggy.git
$ cd bloggy
$ npm install
```

Generate:

``` bash
$ hexo generate
```

Run server:

``` bash
$ hexo server
```
